# Receiver Assembly

[View Materials List](materials.md)

## Steps

### Step 1
Take an 8mm PTFE Tube and insert it into the pivot hole of the Trigger (first and second photos). If it does not fit in you may need to file out the hole and/or adjust printer tolerances. Confirm that a 3/32in by  ¾ in Dowel fits through the inserted tube and can rotate freely (third photo), then remove the 3/32in by  ¾ in Dowel and set it aside. Repeat the same for the Sear (third to sixth photos).

### Step 2
Insert the Trigger into the bottom of the Receiver (first photo), making sure the holes align (second photo). Press a 3/32in by  ¾ in Dowel into the receiver and through the Trigger (third photo), then press the dowel all the way through until it is securely in both sides of the Receiver (fourth photo). Make sure the Trigger rotates freely.

![Step 2 image 1](../../media/receiver/receiver-step-02_picture1.png)
![Step 2 image 2](../../media/receiver/receiver-step-02_picture2.png)
![Step 2 image 3](../../media/receiver/receiver-step-02_picture3.png)
![Step 2 image 4](../../media/receiver/receiver-step-02_picture4.png)

### Step 3
Using the above animation as reference, install a 0.625" Long Compression Spring behind the trigger and the Sear with a 3/32in by  ¾ in Dowel. First, move the Trigger forward and install the 0.625" Long Compression Spring in the circular cutout in the Receiver. Pull the Trigger all the way back so that the Sear can be moved into place so the teeth engage. Once the Sear has been positioned and the hole aligns with the Receiver, push a 3/32in by  ¾ in Dowel into place and let the Trigger relax into its resting position.

![Step 3 image 1](../../media/receiver/receiver-step-03_picture1.gif)

### Step 4
Slide the Grip onto the back of the Receiver (first photo). Screw it in place with (2) 6-32 ⅝”  Phillips Screws (second photo).

![Step 4 image 1](../../media/receiver/receiver-step-04_picture1.png)
![Step 4 image 2](../../media/receiver/receiver-step-04_picture2.png)

### Step 5
Install a 3/32in by  ¾ in Dowel in the top slot on the front of the Receiver (first photo). Slide it in as far as it will go (second photo). If the dowel does not fit, you can carefully bore out the slot with a 3/32 drill bit. The dowel is meant to stay attached to the Receiver, so if it does stay as a friction fit, you can use a drop of superglue to secure it in place. If you are using the ReceiverDualWindow variant, there will be an additional symmetric dowel on the other side (third photo).

![Step 5 image 1](../../media/receiver/receiver-step-05_picture1.png)
![Step 5 image 2](../../media/receiver/receiver-step-05_picture2.png)
![Step 5 image 3](../../media/receiver/receiver-step-05_picture3.png)

### Step 6
Take the TriggerGuard and install the handle side first in the Receiver (second photo), then bend the front side in until it slots into place in front of the Sear (third photo).
