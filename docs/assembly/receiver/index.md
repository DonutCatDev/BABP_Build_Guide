# Receiver Assembly

[View Materials List](materials.md)

## Overview
Assemble the trigger group into the receiver, install the grip and front locator dowel, and snap in the trigger guard.

## Steps

### Step 1: Fit trigger and sear bushings
- Insert an 8 mm PTFE tube into the pivot hole of the Trigger.
- Confirm a 3/32 inch dowel can pass through and rotate freely, then remove the dowel.
- Repeat for the Sear.

![Trigger and sear bushing prep](../../media/other/placeholder.webp)
![Pivot fit check](../../media/other/placeholder.webp)

### Step 2: Install trigger pivot
- Insert the Trigger into the bottom of the Receiver and align the holes.
- Press a 3/32 inch dowel through the receiver and trigger until seated on both sides.
- Verify the Trigger rotates freely.

![Trigger pivot install](../../media/other/placeholder.webp)

### Step 3: Install sear and spring
- Move the Trigger forward and place a 0.625 inch compression spring in the receiver spring pocket.
- Pull the Trigger back, position the Sear so teeth engage, and align pivot holes.
- Press a 3/32 inch dowel through the sear pivot and release the Trigger to resting position.

![Sear and spring install](../../media/other/placeholder.webp)

### Step 4: Install grip
- Slide the Grip onto the back of the Receiver.
- Secure with two 6-32 Phillips screws.

![Grip install](../../media/other/placeholder.webp)

### Step 5: Install front locator dowel
- Install a 3/32 inch dowel in the top front slot of the Receiver.
- Press fully into place.
- If needed, lightly open the slot and use a small drop of superglue only if friction fit is insufficient.
- ReceiverDualWindow variant: install the symmetric dowel on the opposite side.

![Front locator dowel install](../../media/other/placeholder.webp)

### Step 6: Install trigger guard
- Hook the handle side of the TriggerGuard into the Receiver first.
- Flex the front side in until it snaps into the front slot ahead of the Sear.

![Trigger guard install](../../media/other/placeholder.webp)
