# Receiver Assembly Slides

# Step 1
Fit PTFE tube bushings in Trigger and Sear.

# Step 2
Install Trigger and pivot dowel in Receiver.

# Step 3
Install compression spring, then install Sear and pivot dowel.

# Step 4
Install Grip with two 6-32 Phillips screws.

# Step 5
Install front locator dowel (plus second dowel on ReceiverDualWindow).

# Step 6
Snap TriggerGuard into place.
