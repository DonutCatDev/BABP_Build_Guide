# Gear Tensioner Assembly Slides

# Step 1
Prepare InnerGear with 10-32 screw and nut, then glue and cure.

# Step 2
Build FrontConnectorCap gear stack and install washer.

# Step 3
Install 10-32 nut and NutCap in TensionerBlock.

# Step 4
Install 6-32 heat set insert in FrontConnectorBase.

# Step 5
Align and press FrontConnectorBase onto FrontConnectorCap assembly.

# Step 6
Install shroud screw, socket bolts, nuts, lock blocks, and LockBlockCap.

# Step 7
Install TensionerBlock and thumbscrews, verify lock motion.
