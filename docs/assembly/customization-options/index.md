# Customization Options

## Overview
BABP is designed to be modular with multiple customization options. Above are 3 example builds with their associated options.

BABP supports multiple front shroud options. See Shroud Assembly for options and assembly instructions.

Additionally, BABP supports both bolt action (a rotation at the start/end of the prime sequence to unlock/lock the breach in the closed position) and straight pull priming. It does this through a modular set of Receiver, MidBody, and PrimeBlock. Not all are compatible with each other however, and the following table shows what combinations work. First select which PrimeBlock you want (determining the action), then select a compatible Receiver and finally MidBody.

*Technically works but not recommended, as putting a straight pull in a bolt action midbody results in an unused and misleading cutout
