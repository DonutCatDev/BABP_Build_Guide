# Prime Block Assembly Slides

# Common 1
Install front 623 bearing pair with 8 mm M3 screws.

# Common 2
Install rear 623 bearing pair with 8 mm M3 screws.

# Common 3
Set center barrel support bearing and secure.

# Common 4
Install PumpBars and align holes.

# Common 5
Install CatchLever and 3/32 inch dowel.

# Variant
Install front/rear anchor nuts and screws, reinforcement screws, handle heads, and bottom bearing pair per selected variant.
