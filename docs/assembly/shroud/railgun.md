# Railgun Variant

## Overview
This variant uses the RG shroud body and support bars without the integrated bipod assembly.

## Steps

### Step 1: Install Bottom Rail
Install your chosen length of `RG_BottomRail` to `RG_MainBody` with 3x 6-32 Phillips screws.

![railgun-bottom-rail](../../media/shroud/railgun-install-bottom-rail_picture1.png)

### Step 2: Insert Support Bars
Insert the longer `RG_TopSupportBar` and shorter `RG_BottomSupportBar` into the rear slots of `RG_MainBody`.

![railgun-support-bars](../../media/shroud/railgun-insert-support-bars_picture1.png)

### Step 3: Attach Base
Install `RG_Base` to `RG_MainBody` and secure with 5x 6-32 socket head screws. Do not overtighten.

![railgun-attach-base](../../media/shroud/railgun-attach-base_picture1.png)
