# Bipod Variant

## Overview
This variant adds the integrated locking bipod and uses the bipod shroud body components.

## Steps

### Step 1: Install Heat Set Insert
Install 1x 6-32 heat set insert into `RGB_Insert`, flush to the part surface. Insert `RGB_Insert` into `RGB_MainBody` with the flat side up.

![rgb-heat-set-insert](../../media/other/placeholder.webp)

### Step 2: Insert Support Bars and Lock Insert
Insert the longer `RGB_TopSupportBar` and shorter `RGB_BottomSupportBar` into the rear slots. Install 1x 6-32 Phillips screw into the bottom of `RGB_MainBody` to lock the insert and bottom support bar.

![rgb-support-bars](../../media/other/placeholder.webp)

### Step 3: Attach Base
Install `RGB_Base` to `RGB_MainBody` and secure with 5x 6-32 socket head screws. Do not overtighten.

![rgb-attach-base](../../media/other/placeholder.webp)

### Step 4: Install Bipod Sub-Assembly
Slide the completed [Bipod Sub-Assembly](bipod-sub-assembly.md) into the front of `RGB_MainBody`. Use 1x 6-32 socket head screw through the button into the heat set insert. Keep it loose enough to allow swivel; Loctite is recommended.

![rgb-install-bipod-subassembly](../../media/other/placeholder.webp)

### Step 5: Functional Check
Confirm legs extend and lock, lock nubs engage in extended position, and bipod swivels side to side. Adjust screw tension if needed.

![rgb-functional-check](../../media/other/placeholder.webp)
